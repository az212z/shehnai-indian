# مطعم شهناي الهندي — موقع تعريفي (Shehnai Indian Restaurant)

موقع ساكن (Static) فاخر بالعربية RTL لمطعم **شهناي الهندي** في جدة. مبني بـ HTML/CSS/JS فقط بدون أي خطوة بناء، جاهز للنشر على GitHub Pages.

## الرابط الحي (Live URL)
بعد النشر سيكون على:
`https://az212z.github.io/batch9-restaurants/shehnai-indian/`

> حدّث روابط `og:image` و `twitter:image` و JSON-LD `image` في `index.html` إذا تغيّر مسار النشر.

## حالة الـ Backend
- **لا يوجد backend.** نموذج الحجز/الطلب يعمل بالكامل في المتصفح:
  - يتحقق من المدخلات، يحفظ نسخة تجريبية في `localStorage` (مفتاح `shehnai_reservations`)،
  - ثم يفتح **واتساب** على الرقم `966531914513` برسالة عربية مُعبّأة مسبقًا، ويعرض Toast نجاح.
- لا تُرسل أي بيانات إلى أي خادم. الحجز الفعلي يتم عبر واتساب أو الاتصال.

## مصدر الصور
الصور في `assets/img/` هي **صور حقيقية لمطعم شهناي** (واجهة، أطباق، صالة داخلية) + الشعار الرسمي (`sh-5.jpg`) المستخدم كعلامة المطعم (wordmark/emblem).
- `sh-1.jpg` واجهة المطعم ليلاً · `sh-2.jpg` تكا وأرز · `sh-3.jpg` كاري وبرياني · `sh-4.jpg` الصالة الداخلية · `sh-5.jpg` الشعار.

## الموشن التوقيعي
هيرو «الزعفران والبهارات»: نقشة **ماندالا** SVG ترتسم عبر `stroke-dashoffset` وتدور ببطء، ذرّات بهارات زعفرانية تطفو للأعلى بوهج دافئ، وخيط دخان بخور خفيف + Ken Burns على صورة الكاري. يوجد fallback كامل عند `prefers-reduced-motion` (ماندالا مرسومة ثابتة + ذرّات ساكنة).
